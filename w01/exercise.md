# Challenge Besok Pagi — w01

> Kerjakan di **project kerja nyata** kalian (bukan repo latihan). Total ±30 menit.

## Langkah

1. **`/init`** di project kerja → biarkan Claude men-draft CLAUDE.md.
2. **Rapikan** hasilnya pakai template Venturo (`_shared/templates/CLAUDE-go-template.md` atau `CLAUDE-react-template.md`; project lain: ikuti 4 section-nya). Target **≤ 50 baris**.
3. **Plan mode** (`Shift+Tab` 2×) untuk **1 task nyata** dari backlog kalian — review rencananya, edit kalau perlu, baru approve dan biarkan jalan.
4. **`/grademe <nama>`** setelah task selesai (idealnya dari session baru di project yang sama — lihat `grademe/INSTALL.md`).
5. **Screenshot** output /grademe → bawa/kirim sebelum sesi minggu depan.

## Kriteria sukses

- [ ] CLAUDE.md ada di project kerja, ≤ 50 baris, 4 section template terisi
- [ ] Minimal 1 task dieksekusi lewat plan mode dengan rencana yang kalian review (bukan langsung auto-approve)
- [ ] Skor /grademe keluar — berapa pun angkanya (ini baseline, bukan penilaian kinerja; tidak ada target)

## Format lapor (kirim ke channel sesi)

```
Nama:
Project (boleh disamarkan):
Skor /grademe: XX/100
Dimensi terlemah: <nama dimensi> — <1 kalimat kenapa, versi kalian>
1 hal yang mau dicoba beda minggu depan:
+ screenshot output /grademe
```

## Catatan

- Kalau /init menghasilkan draft aneh: itu normal — justru latihan merapikannya adalah intinya.
- Setiap koreksi hari ini, tulis ke CLAUDE.md — jangan cuma perbaiki di chat (koreksi chat = patch sekali-pakai; aturan tertulis = fix selamanya).
- Challenge ini = jalur project EXISTING. Kebetulan mau memulai project BARU? Ikuti alur sesi: tulis definisi produk singkat dulu → `/init` → template → PRD (lihat tabel "Project baru vs existing" di handout) — tetap lapor dengan format yang sama.
- Kalau project kerja sensitif: jalankan di salinan/branch lokal; JANGAN paste screenshot berisi secret (cek sebelum kirim).
- Stuck install /grademe → `grademe/INSTALL.md`, atau tanya di channel.
