# 🌅 Challenge Besok Pagi — w02

> Kerjakan di **project kerja nyata** kalian (bukan `vibescore-api`). Total ±45 menit.
> Deadline: **sebelum sesi w03**.

## Tugas

Pakai **≥ 2 skill baru** dari 8 kartu hari ini, di task nyata dari backlog kerja kalian. Kombinasi bebas — contoh yang biasanya nempel:
- `brainstorming` untuk 1 fitur baru + `/code-review` sebelum PR
- `caveman` sepanjang sesi kerja + `systematic-debugging` saat ketemu bug
- `writing-plans`+`executing-plans` untuk task besar + `ponytail` saat terima diff dari Claude

## Kriteria sukses (bukti, bukan janji)

Sukses kalau kalian punya **minimal salah satu** artefak berikut sebagai bukti skill benar-benar terpakai:
- [ ] File spec dari `brainstorming` (mis. `docs/superpowers/specs/....md`)
- [ ] File plan dari `writing-plans` dengan task `- [ ]` tercentang sebagian/seluruhnya
- [ ] Temuan `/code-review` (screenshot atau tempel daftar temuan — termasuk "no findings" valid)
- [ ] Angka `/caveman-stats` (persen token hemat) dari sesi kerja nyata
- [ ] Ringkasan root cause dari `systematic-debugging` (1-2 kalimat: bug apa, akar masalahnya apa)

Pilih yang relevan dengan skill yang kalian pakai — tidak perlu semua.

## Format lapor (kirim ke kanal sesi, singkat)

```
Skill dipakai : <skill A>, <skill B>
Task          : <1 kalimat>
Nempel?       : <skill mana yang kamu terusin & kenapa>
```

Opsional tapi disarankan: tempel `/grademe <nama-kamu>` harian dari project kerja kalian juga boleh — upload otomatis dedup per `session_id` (aman, tidak akan dobel walau dijalankan berkali-kali dalam sesi yang sama).

## Catatan

- Sebelum install skill APA PUN di luar 8 kartu (mis. dari katalog cadangan handout): baca dulu `SKILL.md`-nya (aturan emas sesi). Lensa cepat: `description`-nya tegas (itu trigger-nya)? ada bagian Gotchas? — tanda skill yang matang.
- `brainstorming` terasa lama/bertele-tele di run pertama? Normal. Time-box sendiri (~10 menit) lalu ketik "cukup, tulis ringkasan spec-nya".
- `caveman` mengubah gaya output seketika — kalau rekan tim kaget lihat output telegrafis, tunjukkan `/caveman-stats`-nya sebagai bukti bukan bug.
- Stuck >30 menit di satu hal → itu juga skill: delegasi. Tanya kanal tim, jangan debug sendirian.
